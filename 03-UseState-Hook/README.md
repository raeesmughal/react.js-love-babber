### Learn UseState hook in react

1. what are hooks in react.js, learn this before useState
2. understand useState
3. practical implementation in counter app


In React, the useState hook is a built-in function that allows you to add and manage "state" (data that changes over time) inside a functional component.

Think of it like a memory for your component. While normal variables "reset" every time a component re-renders, variables created with useState are remembered by React.
